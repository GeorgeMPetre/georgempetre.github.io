OpenCart_Performance_Testing_Project

This project delivers a full end-to-end performance evaluation of an OpenCart e-commerce platform running on a local XAMPP environment. Using Apache JMeter 5.6.3, it assesses system behaviour under load, volume, stress, and scalability scenarios. The test suite includes stepped load tests, concurrency modelling, prolonged stability tests, and stress-to-failure analysis — all supported by PerfMon server monitoring and automated SLA validation (≤3s response time, ≤1% error rate). Each test is documented with clear objectives, load profiles, acceptance criteria, and results, including average response times, throughput, CPU/memory utilisation, disk activity, and pass/fail outcomes. The project also incorporates detailed reporting in XLSX format and professional “Test Not Executed” justifications aligned with industry standards. This work demonstrates proficiency in performance engineering, JMeter scripting, analysis of system bottlenecks, and structured reporting for real-world test scenarios.
